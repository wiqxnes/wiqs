Разберём по шагам: что это такое, когда встречается и как решать каждый вид — без лишних слов.

Общий вид квадратного уравнения
ax2  +  bx  +  c  = 0
a
главный коэффициент, a ≠ 0
b
коэффициент при x
c

💡
Когда уравнение называется неполным?
Если коэффициент b = 0 или c = 0 (или оба сразу) — уравнение неполное. Коэффициент a всегда остаётся, иначе это уже не квадратное уравнение!
Теория

Три вида
неполных уравнений

Смотри на коэффициенты: какой из них пропал — так и определяется вид.

⚡
Запомни правило: смотришь на уравнение — находишь, чего нет (b или c). Это сразу скажет тебе, как решать.
Тип 1
Условие: b = 0 и c = 0
ax² = 0
Оба коэффициента пропали — остался только член с x². Это самый простой случай.
→ x = 0 (один корень)
Тип 2
Условие: b = 0
ax² + c = 0
Нет члена с x. Переносим c вправо, делим на a, извлекаем корень.
→ x = ±√(−c/a)
Тип 3
Условие: c = 0
ax² + bx = 0
Нет свободного члена. Выносим x за скобку — получаем два множителя.
→ x = 0 или x = −b/a

Алгоритм решения

1
Делим обе части на a (a ≠ 0)
ax² = 0 → x² = 0
2
Единственное число, квадрат которого равен 0 — это x = 0
✓
Вывод: уравнение ax² = 0 всегда имеет один корень — x = 0.

Пример

5x² = 0 Тип 1
1
Делим на 5: x² = 0
2
Квадрат равен нулю только когда x = 0
✓ Ответ: x = 0

Алгоритм решения

1
Переносим c в правую часть:
ax² = −c
2
Делим обе части на a:
x² = −c/a
3
Анализируем правую часть:
• если −c/a > 0 → два корня: x = ±√(−c/a)
• если −c/a = 0 → один корень: x = 0
• если −c/a < 0 → корней нет

Примеры

4x² − 9 = 0 Тип 2
1
Переносим: 4x² = 9
2
Делим на 4: x² = 9/4
3
9/4 > 0, берём ±√: x = ±3/2
✓ x₁ = 3/2 ; x₂ = −3/2
6v² + 24 = 0 Нет корней
1
6v² = −24
2
v² = −4 → отрицательное!
✗ Корней нет (v² не может быть −4)

Алгоритм решения

🔑
Главная идея: всегда выносим x за скобку. Произведение двух множителей равно нулю тогда и только тогда, когда хотя бы один из них равен нулю.
1
Выносим x за скобку:
x · (ax + b) = 0
2
Первый множитель: x = 0 — это первый корень
3
Второй множитель: ax + b = 0
Решаем: x = −b/a
✓
Тип 3 всегда даёт два корня: x = 0 и x = −b/a

Пример

3x² − 4x = 0 Тип 3
1
Выносим x: x(3x − 4) = 0
2
x = 0 — первый корень
3
3x − 4 = 0 → x = 4/3
✓ x₁ = 0 ; x₂ = 4/3
